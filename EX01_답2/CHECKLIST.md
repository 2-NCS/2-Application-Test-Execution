# EX01 체크리스트 (제출 전 자가 점검)

## ① 애플리케이션 테스트 수행 (NCS 1)
- [ ] LendServiceTest 의 단위 테스트 5종(정상·재고0 경계·미존재 도서·중복 반납·연체 경계)을 작성했습니다
- [ ] BookRepositoryTest 를 `@DataJpaTest` 로 작성했습니다
- [ ] BookControllerTest 를 `@WebMvcTest` + `@MockitoBean` 으로 작성했습니다(정상 200 / 검증 실패 400)
- [ ] LibraryIntegrationTest 를 `@SpringBootTest` + `MockMvc` 로 작성했습니다(대출→재고 반영)
- [ ] 테스트케이스_명세서 에 각 테스트의 입력·기대결과를 명세했습니다
- [ ] 테스트_실행결과서 에 최초 실행 결과(실패 포함)를 기록했습니다

## ② 애플리케이션 결함 조치 (NCS 2)
- [ ] 실패한 테스트로 드러난 결함을 결함_관리대장 에 기록했습니다(유형·심각도·우선순위·재현 방법)
- [ ] 결함의 원인을 분석하고 src/main 코드를 수정했습니다
- [ ] 수정 후 전체 테스트를 다시 실행하여 모두 통과(GREEN)함을 확인했습니다(회귀 테스트)
- [ ] 결함조치_결과서 에 원인·수정 내용·부작용 없음 확인 결과를 기록했습니다
- [ ] 결함별로 Git 커밋을 남겨 조치 이력을 관리했습니다
